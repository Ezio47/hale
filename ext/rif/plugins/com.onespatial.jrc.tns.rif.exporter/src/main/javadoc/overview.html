The RIF Exporter HALE Plugin library provides functionality to export Humboldt Alignment Editor (HALE) schema mapping definition documents as XML files. These documents are represented internally in the HALE tool in the Geospatial Ontology Mapping Language. The plugin exports them in the W3C Rule Interchange Format - Production Rule Dialect (RIF-PRD) format (see http://www.w3.org/TR/2010/REC-rif-prd-20100622/ for the RIF-PRD specification).

This library is compatible with HALE version 2.0.0.M3.

The starting-point for exploring this library's functionality is the com.onespatial.jrc.hale.ui.RifMappingExportProvider class.

The INSPIRE Schema Transformation Network Service Technical Guidance recommends RIF-PRD as the vendor-neutral schema mapping interchange format to be used to express mappings between data providers' logical schemas and the INSPIRE logical schemas.

The Technical Guidance for the INSPIRE Schema Transformation Network Service is available on the Web at http://inspire.jrc.ec.europa.eu/documents/Network_Services/JRC_INSPIRE-TransformService_TG_v2-0.pdf.
